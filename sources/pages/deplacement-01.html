Défilement

Apprendre à se déplacer sur une page

Objectif : Utiliser la barre de défilement pour lire un texte de haut en bas

💡 La barre de défilement

Sur le côté droit de l'écran, vous voyez une barre verticale grisée avec un rectangle plus foncé. C'est la barre de défilement vertical avec le curseur de déplacement.

Instructions

Comment se déplacer :
Cliquez sur le curseur de déplacement (rectangle foncé) dans la barre grisée sur le bord droit de votre écran.
Maintenir et glisser :
Sans lâcher le clic, déplacez le curseur vers le bas, bien rectiligne, dans la glissière grise.
Analogie :
C'est comme une fermeture éclair : il faut suivre bien d'aplomb le long des dents sans lâcher le curseur.
